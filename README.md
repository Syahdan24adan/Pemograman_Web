# Pemograman_Web

## install node_modules

  `npm install`

## Menginstall Prisma Client 

  `npx prisma generate`
  
## Install Prisma CLI

  `npm install prisma --save-dev`
  
## Migrate schema ke database

  `npx prisma migrate dev --preview-feature`
  
 ## Deploy schema ke database
 
  `npx prisma db push --preview-feature`
